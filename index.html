درباره این دوره

این دوره جامع شما را با مفاهیم اساسی توسعه وب آشنا می‌کند. از HTML ساده شروع کرده و تا ساخت پروژه‌های تعاملی با JavaScript پیش می‌رویم.

ویژگی‌های دوره:

  • ۲۰ درس ساختاریافته و کامل
  • ویرایشگر کد تعاملی با نمایش زنده
  • آزمون‌های خودکار در پایان هر درس
  • مرجع کامل تگ‌های HTML و ویژگی‌های CSS
  • پشتیبانی کامل از راست‌چین و فونت فارسی
  • قابلیت استفاده آفلاین
  • حالت تاریک و روشن

مسیر یادگیری

مرحله اول: HTML

یادگیری ساختار صفحات وب با HTML. از تگ‌های ساده تا عناصر پیچیده و معنایی.

  • ساختار اساسی HTML
  • تگ‌های متنی و قالب‌بندی
  • لینک‌ها، تصاویر و چندرسانه‌ای
  • جداول و فرم‌ها
  • HTML5 و تگ‌های معنایی

مرحله دوم: CSS

طراحی و زیباسازی صفحات وب با CSS. از استایل‌های ساده تا چیدمان‌های پیچیده.

  • انتخابگرها و ویژگی‌های CSS
  • مدل جعبه و موقعیت‌یابی
  • تایپوگرافی و رنگ‌ها
  • Flexbox و CSS Grid
  • طراحی واکنش‌گرا

مرحله سوم: JavaScript

ایجاد تعامل و پویایی در صفحات وب با JavaScript. از مفاهیم پایه تا پروژه‌های عملی.

  • متغیرها، توابع و ساختارهای کنترل
  • آرایه‌ها، اشیاء و دستکاری DOM
  • مدیریت رویدادها و تعامل کاربر
  • اعتبارسنجی فرم و ذخیره‌سازی
  • برنامه‌نویسی ناهمزمان و پروژه نهایی
