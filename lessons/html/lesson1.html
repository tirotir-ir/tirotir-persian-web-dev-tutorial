HTML چیست؟

HTML مخفف HyperText Markup Language است و زبان استانداردی برای ایجاد صفحات وب محسوب می‌شود. HTML ساختار و محتوای صفحات وب را تعریف می‌کند و از تگ‌ها (Tags) برای نشان‌دادن عناصر مختلف استفاده می‌کند.

ویژگی‌های HTML:

  • زبان نشانه‌گذاری ساده و قابل فهم
  • پایه و اساس تمام صفحات وب
  • قابلیت ترکیب با CSS و JavaScript
  • پشتیبانی از تمام مرورگرها
  • استاندارد جهانی W3C

ساختار اساسی HTML

هر سند HTML دارای ساختار مشخصی است که شامل عناصر اصلی زیر می‌باشد:

عناصر اصلی:

  • <!DOCTYPE html> - تعریف نوع سند
  • <html> - عنصر ریشه
  • <head> - اطلاعات متا
  • <body> - محتوای قابل مشاهده

تمرین عملی: اولین صفحه HTML

در ادامه، ساختار اساسی یک صفحه HTML را مشاهده می‌کنید. می‌توانید کد را ویرایش کرده و نتیجه را در نمایش زنده ببینید:

توضیح کد

تگ‌های استفاده شده:

  • <!DOCTYPE html> - به مرورگر می‌گوید که این سند HTML5 است
  • <html lang="fa"> - عنصر ریشه با تعریف زبان فارسی
  • <head> - شامل اطلاعات متا و تنظیمات صفحه
  • <meta charset="UTF-8"> - تعریف کدگذاری کاراکتر
  • <title> - عنوان صفحه که در تب مرورگر نمایش داده می‌شود
  • <body> - محتوای اصلی صفحه
  • <h1> - عنوان اصلی صفحه
  • <p> - پاراگراف متن

تمرین‌های بیشتر

حالا که با ساختار اساسی HTML آشنا شدید، تمرین‌های زیر را انجام دهید:

تمرین ۱:

عنوان صفحه را به "صفحه شخصی من" تغییر دهید.

تمرین ۲:

یک پاراگراف جدید اضافه کنید که در آن خودتان را معرفی کنید.

تمرین ۳:

یک عنوان دوم (h2) با متن "درباره من" اضافه کنید.
